---------------------------------后端加强部分--------------------------------------

在我自己外包工作过程中，最大的感触是，做一个程序除了做页面比较难（属于前端了），就是SQL语句了。因为大部分外包公司，逻辑是非常简单的，框架搭好以后就是复制粘贴了。
