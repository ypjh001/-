	普通调用方法：Outer.Inner oi=new Outer().new Inner();
	静态调用方法：1.Outer.Inner oi=new Outer.Inner();
			      2.Outer.Inner.show();
	内部类用private修饰：
		无法再main函数中调用，怎么办呢？
		在外部类中创建一个方法，用来判定是否可以调用此内部类。
		class Outer{
			private class Inner{
				public void show(){}
			}
			
			public void use(){
				if(被允许){
					Inner in=new Inner();
					in.show();
				}
			}
		}

局部内部类
	面试题：
		局部内部类访问局部变量的注意事项？
			局部变量必须用final修饰
			因为，局部变量会随着方法的调用而调用，方法调用完毕，就会消失
			而局部内部类属于堆内存，不会立即消失，所以要用final修饰，变成
			常量，就算消失了，在内存中存储的数据还可以使用。
